Welcome email setup

The welcome email is sent automatically via a Loops transactional email whenever a new subscriber signs up. Set up the trigger in Loops, then use this page to manage and preview the template content.

📋 How to set this up in Loops

  1. In Loops, go to Transactional → Create transactional email
  2. Design your welcome email template (use the preview below as a guide)
  3. Copy the Transactional ID and paste it into the API Config panel above
  4. When someone subscribes on gobebop.app, call the Loops API with their email to trigger the send
```

How to send the welcome email

To send the welcome email to a new subscriber, copy the HTML above and create a new campaign in Mailchimp addressed to that person. In the future, this can be automated via your signup form — ask your husband to trigger the welcome send when a new subscriber is added.

Steps to send in Mailchimp:

  1. Click "👁 Preview welcome email" then "📋 Copy HTML for Mailchimp" above
  2. In Mailchimp, create a new campaign → Single email
  3. Under Design, choose "Code your own" and paste the HTML
  4. Set the recipient to the new subscriber's email
  5. Send immediately
